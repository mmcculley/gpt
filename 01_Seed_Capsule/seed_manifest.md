# Seed Capsule Manifest

This folder defines the foundational identity and rebuild instructions for the Voice GPT Framework. It contains the system prompt, voice persona, recovery protocols, and conversational anchoring styles. These files ensure the project can be reinstated even after memory or context loss.

## Files Included:
- `system_prompt.md`: Core logic, tone, tools, and recovery behaviors
- `persona_manifest.md`: Voice, user preferences, and quoting resolution
- `rebuild_protocol.md`: How to reload the project into a new GPT instance
- `conversation_hooks.md`: Signature triggers and style anchors
